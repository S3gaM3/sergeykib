## Аватар

Файл из `person.avatar` в `content.tsx` (путь в `public`, например `/assets/img/portfolio/avatar.webp`).

## Favicon

Иконка в `public` / настройки Next — см. актуальную структуру репозитория.

### Персоналия

Имя, роль, аватар, `location` (IANA), `languages` — для страницы «Обо мне» и шапки (если включено).

### Соцсети

Массив `social`: `name`, `icon`, `link`. Иконки — в [`src/resources/icons.ts`](src/resources/icons.ts), см. [документацию Once UI](https://once-ui.com/docs/icons).

<CodeBlock
    marginBottom="16"

    { name: "GitHub", icon: "github", link: "https://github.com/..." },
];`,
    language: "tsx",

  }
]} />

### Страницы

Объекты `home`, `about`, `blog`, `work`, `gallery`: `label` (навигация), `title`, `description` (meta/OG), плюс поля страницы.
