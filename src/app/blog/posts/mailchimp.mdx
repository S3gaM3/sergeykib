## Аккаунт

Нужен список рассылки в [Mailchimp](https://mailchimp.com) и [встроенная форма](https://mailchimp.com/help/add-a-signup-form-to-your-website/) — URL действия формы.

## Конфиг

В [`src/resources/once-ui.config.ts`](src/resources/once-ui.config.ts) объект `mailchimp`: `action` — URL post из встраивания.

<CodeBlock
    highlight="2"
    codes={[
  {
    code:
`const mailchimp: MailchimpConfig = {
    action: "https://.../subscribe/post?...",
    effects: { /* фон блока */ },
};`,
    language: "tsx",
    label: "src/resources/once-ui.config.ts"
  }
]} />

## Тексты блока

В [`src/resources/content.tsx`](src/resources/content.tsx) объект `newsletter`: `display`, `title`, `description`. При `display: false` компонент [`Mailchimp`](src/components/Mailchimp.tsx) не рендерится.

<CodeBlock
    highlight="2"
    codes={[
  {
    code:
`const newsletter: Newsletter = {
    display: false,
    title: <>Заголовок</>,
    description: <>Текст под заголовком</>,
};`,
    language: "tsx",
    label: "src/resources/content.tsx"
  }
]} />

## Оформление блока

Поле `mailchimp.effects` задаёт маску/градиент/точки для фона секции рассылки (аналогично общему `effects`).
