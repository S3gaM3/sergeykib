## Список путей

В [`src/resources/once-ui.config.ts`](src/resources/once-ui.config.ts) объект `protectedRoutes`: ключ — путь, значение `true`, если нужен пароль.

<CodeBlock
    marginBottom="16"
    highlight="2"
    codes={[
  {
    code:
`const protectedRoutes: ProtectedRoutesConfig = {
  "/work/secret-slug": true,
};`,
    language: "tsx",
    label: "src/resources/once-ui.config.ts"
  }
]} />

[`RouteGuard`](src/components/RouteGuard.tsx) запрашивает пароль, если есть рабочие `/api/check-auth` и `/api/authenticate`.

## Переменные окружения

Пароль задаётся в `.env.local` и на хостинге. При `output: "export"` API-маршрутов нет — защита через форму не сработает; это отражено в UI RouteGuard.
