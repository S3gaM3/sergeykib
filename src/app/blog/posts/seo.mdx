## Meta-теги

`Meta.generate` и&nbsp;страницы подставляют title и&nbsp;description из [`src/resources/content.tsx`](src/resources/content.tsx) и [`once-ui.config.ts`](src/resources/once-ui.config.ts) (аналог старых `content.js` / `config.js`).

## Schema

JSON-LD собирается компонентом `Schema` из Once UI на&nbsp;основе тех&nbsp;же конфигов.

## Open Graph

При **static export** (`output: "export"`) маршрута `/api/og` нет. Берите файл из `public` и абсолютный URL через `toAbsoluteUrl(baseURL, path)` ([`src/utils/absoluteUrl.ts`](src/utils/absoluteUrl.ts)).
