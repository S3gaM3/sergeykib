## Файлы постов

Создание, правка и удаление: добавляйте или убирайте `*.mdx` в `src/app/blog/posts`.

## Frontmatter

Блок `---` в начале файла задаёт title, summary, image, дату и теги для страницы поста и для meta/OG.

<CodeBlock
    marginBottom="16"
    codes={[
      {
        code:
`---
title: "Заголовок поста"
publishedAt: "2024-04-08"
image: "/images/gallery/img-02.jpg"
summary: "Краткое описание для списка и соцсетей."
tag: "Тег"
---`,
        language: "tsx",
        label: "src/app/blog/posts/post-1.mdx"
      }
    ]}
/>

## Синтаксис MDX

Контент поста — MDX. Часть элементов подменяется компонентами Once UI через `src/components/mdx.tsx`.

## Свои компоненты

Новые теги MDX: импорт в `src/components/mdx.tsx` и запись в объект `components`.

<CodeBlock
    marginBottom="16"
    highlight="11-12"
    codes={[
      {
        code:
`const components = {
    p: createParagraph as any,
    h1: createHeading(1) as any,

    img: createImage as any,
    a: CustomLink as any,
    Table,
    CodeBlock,
};`,
        language: "tsx",
        label: "src/components/mdx.tsx"
      }
    ]}
/>

`Table` и `CodeBlock` уже подключены; остальное — по тому же принципу.

## Перезагрузка

Горячая перезагрузка MDX в dev может отсутствовать; при правках — обновление страницы вручную.
