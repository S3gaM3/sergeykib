## Глобальные токены

Тема и палитра задаются объектом `style` в [`src/resources/once-ui.config.ts`](src/resources/once-ui.config.ts) и провайдерами в [`src/components/Providers.tsx`](src/components/Providers.tsx). Подбор цветов: [Once UI Customize](https://once-ui.com/customize).

<CodeBlock
    marginBottom="16"
    codes={[
  {
    code:
`const style: StyleConfig = {
  theme: "system",
  neutral: "gray",
  brand: "cyan",
  accent: "red",
  solid: "contrast",
  solidStyle: "flat",
  border: "playful",
  surface: "translucent",
  transition: "all",
  scaling: "100",
};`,
    language: "tsx",
    label: "src/resources/once-ui.config.ts"
  }
]} />

## Фон в layout

Градиент, сетка, точки и линии — объект `effects` в том же файле; в [`src/app/layout.tsx`](src/app/layout.tsx) передаётся в `<Background />`.

<CodeBlock
    codeHeight={24}
    marginBottom="16"
    codes={[
  {
    code:
`const effects: EffectsConfig = {
    mask: { cursor: false, x: 50, y: 0, radius: 100 },
    gradient: { display: false, opacity: 100, /* ... */ },
    dots: { display: false, opacity: 40, /* ... */ },
    grid: { display: false, opacity: 100, /* ... */ },
    lines: { display: false, opacity: 100, /* ... */ },
};`,
    language: "tsx",
    label: "src/resources/once-ui.config.ts"
  }
]} />
