## Обзор

Небольшой продуктовый проект для управления пользовательскими cookie-настройками.

## Что сделано

- Реализован UI для выбора и&nbsp;сохранения cookie-политики.
- Учтены базовые требования по безопасности и управлению согласием.
- Подготовлена архитектура для расширения категорий cookie.

## Стек

JavaScript, HTML5, CSS3, Web APIs.
