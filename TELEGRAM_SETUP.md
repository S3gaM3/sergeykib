## Шаг 1: Создание бота (уже выполнено)

1. Откройте Telegram и найдите бота [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям: укажите имя и username для бота
4. BotFather выдаст токен вида: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`
5. Скопируйте токен

## Шаг 2: Получение Chat ID

Есть несколько способов:

### Способ 1: Использование @userinfobot
1. Найдите в Telegram бота [@userinfobot](https://t.me/userinfobot)
2. Начните с ним диалог командой `/start`
3. Бот отправит ваш Chat ID

### Способ 2: Использование @RawDataBot
1. Найдите бота [@RawDataBot](https://t.me/RawDataBot)
2. Начните диалог командой `/start`
3. Бот отправит информацию в формате JSON, найдите поле `"id"` - это ваш Chat ID

### Способ 3: Получить через API (после создания бота)
1. Отправьте сообщение вашему боту
2. Перейдите по ссылке: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
3. Найдите в ответе поле `"chat":{"id":...}` - это ваш Chat ID

## Шаг 3: Тестирование (Форма уже готова!)

1. Запустите проект: `npm run dev`
2. Откройте сайт и найдите форму в разделе "Контакты"
3. Заполните форму и отправьте тестовое сообщение
4. Проверьте, что сообщение пришло в Telegram

## Примечания

### ⚠️ Важно: Ограничения GitHub Pages

Проект настроен на статический экспорт (`output: 'export'` в `next.config.js`). Это означает:
- API routes **не будут работать** при деплое на GitHub Pages
- Форма будет работать только в режиме разработки (`npm run dev`) или на платформах с поддержкой серверных функций

### Варианты решения для продакшена:

1. **Vercel** (рекомендуется для Next.js)
   - Просто подключите репозиторий к Vercel
   - API routes будут работать автоматически
   - Бесплатный план включает 100GB bandwidth

2. **Netlify**
   - Поддерживает Next.js API routes
   - Настройте через Netlify CLI или веб-интерфейс

3. **Отдельный backend-сервер**
   - Вынесите API route в отдельный сервис (Node.js/Express)
   - Настройте CORS для работы с фронтендом

4. **Использование внешних сервисов**
   - Formspree, FormSubmit и другие сервисы для форм
   - Настройте webhook для отправки в Telegram
